Установка на Windows
1
Скачай архив с Mac
Файл sergio-windows-transfer.zip лежит на рабочем столе Mac. Перенеси через USB, Telegram (себе в Избранное) или Google Drive.
2
Открой PowerShell
Win + R → введи powershell → Enter
3
Запусти одну команду
Если архив в папке Downloads:
# Вставь в PowerShell и нажми Enter
Set-ExecutionPolicy Bypass -Scope Process -Force; $z="$env:USERPROFILE\Downloads\sergio-windows-transfer.zip"; $d="$env:USERPROFILE\Downloads\sergio-setup"; Expand-Archive $z -DestinationPath $d -Force; & "$d\sergio-windows-transfer\windows-install.ps1"
⚠️ Если архив не в Downloads — измени путь после $z=
4
Авторизуй Claude Code
После установки в PowerShell набери claude — откроется браузер, войди в аккаунт Anthropic.
5
Открой проекты в VS Code
code %USERPROFILE%\Projects\ClaudeProjects
Что установится автоматически: Git · Node.js · Python 3 · VS Code · Claude Code · все 14 скиллов · память Claude · все проекты

Mac vs Windows — разделение задач

🍎 Mac (основной)

  • iOS сборка (Xcode + Capacitor)
  • Мобильный дашборд автозапуск
  • iCloud синхронизация (Scriptable)
  • Claude метрики (JSONL локально)
  • Git push в основных ветках

🪟 Windows (второй)

  • Веб-разработка (HTML/CSS/JS)
  • Excel финмодели
  • Claude Code — все проекты
  • Android (через Codemagic CI)
  • Мобильный дашборд вручную
