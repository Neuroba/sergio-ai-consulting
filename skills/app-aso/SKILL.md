# App ASO — оптимизация для App Store / Google Play

## Когда использовать

- Подготовка к публикации в сторе
- «Напиши описание для App Store»
- Подбор ключевых слов, категории
- Создание скриншотов для стора

## Инструкции

### Шаг 1: Метаданные приложения

Заполни по шаблону `tools/aso-template.json`:
- **Title** (30 символов max iOS, 50 Android) — бренд + ключевое слово
- **Subtitle** (30 символов iOS) — главная выгода
- **Keywords** (100 символов iOS, через запятую без пробелов)
- **Short description** (80 символов Android)
- **Full description** (4000 символов) — структура в `tools/description-structure.md`
- **Category** — выбери primary + secondary
- **Content rating** — возрастной рейтинг

### Шаг 2: Визуалы

| Элемент | iOS | Android |
|---------|-----|---------|
| Иконка | 1024×1024 PNG, без альфа | 512×512 PNG |
| Скриншоты | 6.7" (1290×2796), 5.5" (1242×2208) | 16:9 или 9:16, min 320px |
| Количество | 3-10 | 2-8 |
| Видео | 15-30 сек, опционально | до 30 сек |

Скриншоты: не голый UI → фреймы устройства + заголовки + фон.
Используй `tools/screenshot-prompt.md` для генерации через AI.

### Шаг 3: Ключевые слова (iOS)

1. Основное слово (то, что ищут) — в Title
2. 3-5 высокочастотных в Keywords
3. Длинный хвост — в описании и subtitle
4. НЕ повторяй слова из Title в Keywords (Apple дублирует сам)
5. Без пробелов после запятых: `отжимания,фитнес,тренировка,счётчик`

### Шаг 4: Локализация

Минимум: RU + EN + ES (для Аргентины).
Каждый язык — отдельные title, subtitle, keywords, description.
Используй скилл `rioplatense-copy` для ES-версии.

## Принципы

- Первые 3 скриншота решают 80% конверсии
- Title: бренд + одно ключевое слово (не спам)
- Описание: первые 3 строки видны без «ещё» — вся суть там
- Обновляй ключевые слова раз в 2-4 недели по аналитике
