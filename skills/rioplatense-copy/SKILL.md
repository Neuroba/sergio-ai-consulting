# Rioplatense Copy — копирайтинг на аргентинском испанском

## Когда использовать

- Тексты для PicadaRica, Comida Smart или любого бизнеса в Буэнос-Айресе
- Посты для Instagram / Facebook на испанском
- Вакансии для ZonaJobs / Facebook
- Лендинги, описания продуктов
- Перевод RU → ES (аргентинский)

## Инструкции

### Тон и стиль

1. **Voseo обязателен:** «vos» вместо «tú», спряжения: «tenés», «podés», «sabés», «querés», «hacé», «mirá», «contanos»
2. **Rioplatense лексика:**
   - «bárbaro» (отлично), «piola» (умный/крутой), «copado» (классный)
   - «laburo» (работа), «guita» (деньги), «posta» (правда/реально)
   - «re» как усилитель: «re loco», «re bueno»
   - «che» как обращение
   - «dale» (давай/ок)
   - «de una» (сразу/конечно)
3. **Избегать:**
   - Мексиканизмов: «güey», «chido», «mero», «padre» (в значении «круто»)
   - Испанского tuteo: «tú tienes», «tú puedes»
   - Англицизмов без нужды (но «ok», «cool» допустимы в неформальном)
4. **Формат обращения:**
   - B2C неформальный: voseo, «che», прямое обращение
   - B2B: более формальный, но всё равно voseo (никогда tuteo)

### Форматы

| Формат | Длина | Особенности |
|--------|-------|-------------|
| Instagram caption | 150-300 слов | Хуки, эмодзи умеренно, CTA в конце, хештеги |
| Facebook post | 100-200 слов | Больше текста, меньше хештегов |
| Vacante (ZonaJobs) | Структурированная | Zona/Rubro/Modalidad, preguntas de screening |
| Landing page | По секциям | H1 → subtitle → CTA, pain/solution/benefit |
| WhatsApp mensaje | 2-3 строки | Кратко, прямо, CTA |

## Принципы

- Аргентинский испанский — это НЕ перевод с кастильского. Это отдельный голос.
- Никогда не используй «tú». Даже случайно.
- Цены: в USD знак $ перед числом, в ARS — тоже $, но уточняй «en pesos».
- Числа: запятая для тысяч (1.000), точка для десятичных (3,5) — аргентинский формат.
