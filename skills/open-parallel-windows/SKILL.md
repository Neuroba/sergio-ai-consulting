# Open Parallel Windows — открыть N окон VSCode

## Когда использовать
- «открой ещё окно», «создай окна», «параллельно», «ещё одно окно для X»
- Нужно работать одновременно в 2+ проектах
- Запуск новой задачи в отдельном Claude Code контексте

## Инструкции

### Шаг 1: Определи N
По умолчанию N=2. Если в запросе есть цифра или список проектов — используй её.

### Шаг 2: Запусти скрипт
```bash
bash ~/.claude/skills/open-parallel-windows/tools/open_windows.sh <N>
```

### Шаг 3: Подтверди
Скажи сколько окон открыто. Напомни в каком проекте работать в каждом окне (если пользователь указал).

## Правила
- Всегда открывать `ClaudeProjects.code-workspace` — даёт доступ ко всем 7 проектам
- Если проект не указан — открыть N одинаковых окон с workspace
- Если указан проект — напомнить пользователю перейти в нужную папку внутри окна
