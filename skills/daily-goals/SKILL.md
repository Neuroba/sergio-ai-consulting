# Daily Goals — ежедневный запуск целей

## Когда использовать

- Каждое утро перед работой
- Sergio говорит: «цели», «утро», «план дня», «что сегодня по целям»
- Нужно согласовать фокус дня с долгосрочными целями

---

## ЦЕЛИ (зафиксированы, проверять через эти линзы)

```
ЦЕЛЬ 1: ЗДОРОВЬЕ (фундамент)
  → Вес: 52 кг → 60 кг (дефицит массы)
  → Трезвость с 18.02.2026 (считать дни)
  → Камни в почках: пить воду, ограничить оксалаты
  → Без бега до травматолога
  → Курение: снижать
  → Ежедневно: белок + калории + активность

ЦЕЛЬ 2: ФИНАНСЫ (дедлайн: июнь 2027)
  → Таргет: $5000/мес
  → Сейчас: ~$294/мес (PicadaRica, май 2026)
  → Путь: PicadaRica масштаб + PushUp монетизация + новые потоки
  → Каждый день — что сдвинулось к $5000?

ЦЕЛЬ 3: ОБОРУДОВАНИЕ (когда финансы позволят)
  → Новый MacBook (Air 2015 — потолок)
  → Покупать только когда доход стабилен
```

---

## Инструкции (выполнять строго по шагам)

### Шаг 1: Параллельно загрузить данные

Запусти одновременно:
1. **Todoist** — `find-completed-tasks` за вчера (дату считать от сегодня -1 день)
2. **Todoist** — `find-tasks-by-date` на сегодня (что запланировано)
3. **Notion** — `notion-fetch` Dashboard: `38a6a70c-e737-8139-9d56-e3bcb4e9ce8b`

### Шаг 2: Показать СНИМОК ЦЕЛЕЙ

Вывести в формате:

```
## Цели — [дата], день [N] трезвости

### 1. ЗДОРОВЬЕ
Трезвость: [N] дней ✓
Вчера: [если есть данные из Todoist — что сделал для здоровья]
Напоминание: [1 конкретное — вода/белок/активность — выбрать самое важное]

### 2. ФИНАНСЫ → $5000/мес (до июня 2027)
Сейчас: ~$294/мес | Осталось: [X]$ | Прогресс: [%]
Вчера по бизнесу: [из Todoist выполненных]

### 3. ОБОРУДОВАНИЕ
Статус: ждём — доход должен вырасти до [порог]

---
### 3 вопроса для согласования:
```

### Шаг 3: ЗАДАТЬ 3 ВОПРОСА (обязательно, не пропускать)

После снимка задай три вопроса — по одному блоку:

```
**А. ЗДОРОВЬЕ вчера:**
Что поел / сколько воды / была ли активность?
(можно одной строкой)

**Б. БИЗНЕС вчера:**
Что продвинулось? Были ли продажи/контакты/деньги?
(проект + факт)

**В. ФОКУС СЕГОДНЯ:**
1 задача на здоровье + 1 задача на деньги. Что выбираешь?
```

### Шаг 4: ПОСЛЕ ОТВЕТОВ — план дня

Когда Sergio ответил на вопросы:

1. **Оцени** — соответствует ли его фокус целям? Если нет — мягко скажи
2. **Создай задачи** в Todoist (tool: `add-tasks`):
   - Задача по здоровью на сегодня (то что он сказал)
   - Задача по бизнесу на сегодня (то что он сказал)
3. **Выведи итоговый план:**

```
## Согласовано на [дата]

ЗДОРОВЬЕ: [задача] → создано в Todoist ✓
БИЗНЕС:   [задача] → создано в Todoist ✓

Трезвость: [N] дней. [короткое поощрение или факт]

---
Приоритет дня: [1 предложение — что главное]
```

---

## Правила

- Дни трезвости считать от 18.02.2026 до сегодня
- Не задавай лишних вопросов — только 3 из Шага 3
- Если Sergio пропускает вопрос — не переспрашивай, двигайся дальше
- Снимок целей всегда показывать ПЕРЕД вопросами
- Не добавляй задачи без его ответов — только после согласования
- Прогресс к $5000: считать (5000 - текущий_доход) / 5000 * 100%
