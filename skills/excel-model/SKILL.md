# Excel Model — генератор финансовых моделей

## Когда использовать

- «Сделай Excel / табличку / spreadsheet»
- «Как для покерного клуба» (отсылка к предыдущим моделям)
- Трекер продаж, калькулятор прибыли, P&L за день/неделю
- Любая задача, где нужен .xlsx с формулами

## Инструкции

### Шаг 1: Определи тип модели

Поддерживаемые шаблоны:
- **sales-tracker** — трекер дневных продаж (вводишь штуки → считает выручку, зарплату, прибыль)
- **pnl** — P&L за период (доходы, расходы, маржа)
- **custom** — произвольная модель

### Шаг 2: Собери параметры

- Какие колонки / строки нужны
- Какие формулы (суммы, проценты, условные)
- Нужна ли условная подсветка (план достигнут / не достигнут)
- Валюта (USD / ARS / другая)
- Куда сохранить файл

### Шаг 3: Запусти генератор

```bash
python3 ~/.claude/skills/excel-model/tools/generate.py \
  --type sales-tracker \
  --products '[{"name":"50g","price":8,"cost":2.5},{"name":"250g","price":35,"cost":12.5}]' \
  --fixed-costs 76 \
  --seller-base 20 \
  --commission-rate 0.25 \
  --plan-revenue 590 \
  --output "/путь/к/файлу.xlsx"
```

### Шаг 4: Покажи пользователю

- Скажи куда сохранён файл
- Объясни какие ячейки вводить (жёлтые), какие считаются сами (серые)
- Предложи открыть в Numbers / Excel / Google Sheets

## Принципы

- Жёлтые ячейки = ввод данных, серые = формулы (не трогать)
- Подсветка: зелёный = план выполнен, красный = нет
- Формулы на английском (SUM, IF) — работают везде
- Всегда добавлять лист "Инструкция" с пояснениями на русском
